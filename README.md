A modern, fully responsive web layout built using **pure HTML & CSS**.\
This project demonstrates **mobile-first design**, **fluid layouts**,

## 📌 Project Overview

**FlowSpace** is a frontend project focused on building a scalable and
responsive UI system that adapts seamlessly across devices --- from
small mobile screens to large desktops.

------------------------------------------------------------------------

## 🛠️ Tech Stack

-   HTML5
-   CSS3
-   CSS Grid
-   Flexbox
-   Media Queries
-   Popover API
-   Fluid Typography (clamp)

------------------------------------------------------------------------

## ✨ Features

-   Fully responsive (mobile to desktop)
-   Mobile-first design
-   Grid + Flexbox layout
-   Fluid typography
-   No JavaScript navigation (Popover API)
-   Accessibility-friendly
-   Lightweight and fast

------------------------------------------------------------------------

## 📂 Project Structure

FlowSpace/ │── index.html\
│── style.css\
│── script.js\
│── README.md

------------------------------------------------------------------------

## 📖 Learning Outcome

-   Responsive design principles\
-   Clean CSS architecture\
-   Modern layout techniques\
-   Accessibility basics

------------------------------------------------------------------------

## 🧪 How to Run

1.  Download or clone the repo\
2.  Open `index.html` in your browser
